# smartedu-dl (`go`)

> 智慧教育平台资源下载工具（`go`实现版，仅基于 fyne 框架 GUI 版本，正在进行中）

## 📝 功能说明

主要支持`smartedu.cn`教材、课件（PDF 格式）下载存储。
